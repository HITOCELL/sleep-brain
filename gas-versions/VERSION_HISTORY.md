# GAS_CODE バージョン履歴

## 戻し方
1. 戻したいバージョンのファイルを開く
2. 全コピー
3. GASエディタに貼り付け → 新しいバージョンでデプロイ

---

## v8（現在の最新）— 2026-05-03
**ファイル:** GAS_CODE_v8.js  
**変更点:**
- **サマリー即時反映**: `handleSubmit` と `handleTracking` の末尾で `buildSummary()` を呼ぶよう変更（データ受信のたびにサマリーが即時更新）
- **離脱率計算変更**: 離脱率 = 各ステップの離脱数 ÷ 全体の総離脱数 × 100（全ステップ合計が100%になる方式）
- **色分け変更**: 総離脱数に占める割合で色分け（20%以上:赤 / 12%以上:オレンジ / 7%以上:黄 / 3%以上:黄緑 / それ以下:緑）
- 離脱数列と離脱率列を同じ色で表示
